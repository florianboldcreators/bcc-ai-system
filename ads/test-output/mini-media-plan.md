## 1. Ad Copy Varianten — Hook-Body-CTA Framework

Basierend auf dem genehmigten GRWM-Konzept "Mein letztes Accessoire":

---

### V1 — Pain Point

| Element | Copy |
|---------|------|
| **HOOK** | "Dein Outfit ist fertig — aber irgendwas fehlt." |
| **BODY** | Jacke sitzt. Sneaker fresh. Aber das beste Piece steht noch draußen. Der neue MINI Cooper. Klein, aber Hauptcharakter. |
| **CTA** | "Jetzt konfigurieren." |

**Rationale:** Spricht das Gefühl an, dass ein Look nie "komplett" ist. Die Auflösung — das Auto als fehlendes Accessoire — überrascht und verankert die Positionierung.

---

### V2 — Social Proof

| Element | Copy |
|---------|------|
| **HOOK** | "Warum 3 von 4 MINI-Fahrern unter 30 sind." |
| **BODY** | Kein SUV-Flex. Kein Papaauto. Der Cooper ist das Auto, das zu deinem Style passt — nicht umgekehrt. |
| **CTA** | "Finde deinen Style." |

**Rationale:** Demographische Wahrheit als Social Proof. Erzeugt In-Group-Gefühl bei Gen Z — "die anderen in meinem Alter fahren das auch."

---

### V3 — Curiosity Gap

| Element | Copy |
|---------|------|
| **HOOK** | "Was alle auf der Straße anstarren — und es ist nicht du." |
| **BODY** | British Racing Green. Runde Haube. Kopfsteinpflaster-Energie. Der neue Cooper stiehlt dir den Look — und du liebst es. |
| **CTA** | "Entdecke den Cooper." |

**Rationale:** Neugier-Gap funktioniert auf TikTok mit 85%+ Scroll-Stop-Rate. Die Implikation "nicht du, sondern dein Auto" ist charmant provokant.

---

### V4 — Bold Statement

| Element | Copy |
|---------|------|
| **HOOK** | "Dieses Auto hat mehr Personality als dein Ex." |
| **BODY** | Kompakt. Eigenständig. Macht Spaß und ghostet dich nicht. Der MINI Cooper — dein zuverlässigstes Accessoire. |
| **CTA** | "Jetzt Probefahrt buchen." |

**Rationale:** Bewusst polarisierend. TikTok-Kommentarsektionen lieben kontroverse Takes — generiert organische Reichweite durch Diskussion.

---

### V5 — UGC Style

| Element | Copy |
|---------|------|
| **HOOK** | "pov: du fährst zum ersten Mal den Cooper und dein ganzer Feed ändert sich" |
| **BODY** | Kein Script. Kein Filter. Einfach einsteigen, losfahren, Grinsen im Rückspiegel. So fühlt sich Hauptcharakter-Energie an. |
| **CTA** | "Link in Bio." |

**Rationale:** Passt 1:1 zum GRWM-Format des genehmigten Konzepts. Niedrige Produktionsbarriere, hohe Authentizität. Kann auch als Spark Ad mit echten Creatorn skaliert werden.

## 3. A/B Tests — 3 Hypothesen

### Test 1: Hook-Typ

| Parameter | Details |
|-----------|---------|
| **Hypothese** | UGC-Style Hooks (V5) erzielen eine 25% höhere Watch-Through-Rate als polierte Brand-Hooks (V2), weil TikTok-Nutzer Authentizität über Produktion priorisieren. |
| **Variante A** | V5 — "pov: du fährst zum ersten Mal..." (UGC, unpolished) |
| **Variante B** | V2 — "Warum 3 von 4 MINI-Fahrern..." (Brand-Voice, poliert) |
| **KPI** | Watch-Through-Rate (75%+), 6-Sekunden-View-Rate |
| **Sample Size** | Min. 50.000 Impressions pro Variante |
| **Laufzeit** | Woche 1–2 |
| **Budget** | €1.500 (€750 pro Variante) |
| **Entscheidung** | Gewinner-Hook wird für Woche 3–4 skaliert |

---

### Test 2: Targeting — Interest vs. Lookalike

| Parameter | Details |
|-----------|---------|
| **Hypothese** | Lookalike Audiences (basierend auf Konfigurator-Besuchern) liefern einen 30% niedrigeren CPA als Interest-Based Targeting, weil Intent-Signale stärker sind als deklarierte Interessen. |
| **Variante A** | Interest Targeting: Mode + Auto + Lifestyle Interessen |
| **Variante B** | Lookalike 1% auf Konfigurator-Besucher |
| **KPI** | CPA (Klick auf Konfigurator), CTR, CPM |
| **Sample Size** | Min. 30.000 Impressions pro Variante |
| **Laufzeit** | Woche 1–2 |
| **Budget** | €1.500 (€750 pro Variante) |
| **Entscheidung** | Effizientere Audience wird mit 60% des Budgets in Woche 3–4 bespielt |

---

### Test 3: CTA-Typ — Soft vs. Hard Conversion

| Parameter | Details |
|-----------|---------|
| **Hypothese** | Ein weicher CTA ("Entdecke den Cooper") erzielt 40% mehr Klicks als ein harter CTA ("Jetzt Probefahrt buchen"), da Gen Z in der Awareness-Phase keine Commitment-CTAs akzeptiert. |
| **Variante A** | Soft CTA — "Entdecke den Cooper" → Landing Page |
| **Variante B** | Hard CTA — "Jetzt Probefahrt buchen" → Buchungsformular |
| **KPI** | CTR, Bounce Rate auf Landing Page, Micro-Conversions |
| **Sample Size** | Min. 40.000 Impressions pro Variante |
| **Laufzeit** | Woche 2–3 |
| **Budget** | €1.000 (€500 pro Variante) |
| **Entscheidung** | Gewinner-CTA wird auf alle Ad-Varianten angewendet |

### Spend-Regeln
- **Kill-Regel:** Variante mit CTR <0,8% nach 10.000 Impressions wird pausiert
- **Scale-Regel:** Variante mit CTR >2% und CPM <€8 bekommt +50% Budget
- **Daily Cap:** Max. €400/Tag um Audience Fatigue zu vermeiden
- **Frequency Cap:** Max. 3x pro User pro Woche
