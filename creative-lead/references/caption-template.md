**Character count:** [X] / 300

```

```

### Native German Quality Check
- [x] Verb position correct (V2 main clause, verb-final subclauses)
- [x] No literal English idiom translations
- [x] Short sentences, natural rhythm
- [x] Reads naturally when spoken aloud

---

## Formatting Rules
- **Instagram hashtags:** 8-12, placed AFTER caption text, separated by line break
- **TikTok hashtags:** 3-5, placed AFTER description, no line break needed
- **Emoji usage:** Moderate — 2-4 per caption, never in hashtags
- **Line breaks:** Use for readability, especially on Instagram
- **CTA:** Always include one (comment prompt, save prompt, share prompt, link reference)

## Character Limits
| Platform | Format | Max Characters |
|----------|--------|---------------|
| Instagram | Reels | 1,000 |
| Instagram | Carousel | 1,500 |
| Instagram | Static | 1,500 |
| TikTok | All | 300 |
