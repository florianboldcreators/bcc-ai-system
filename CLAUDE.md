# BCC AI System

This repo contains AI role clones for Bold Creators Club (BCC), a content/video agency in Munich.

## Mission
Transform BCC from 40-person agency to 5-person AI-augmented company. Each role clone is a Skill that can run on Claude.ai, Claude Code, or API.

## Current Clones
- **Creative Lead** — `.claude/skills/creative-lead/` — Develops creative concepts for social campaigns
- **CPTO** — `cpto/` — Oversees the AI transformation strategy

## Knowledge Base
- `creative-lead/knowledge-base/` — Slack exports (52K lines), Asana tasks (64), brand voices, examples
- Brand voice profiles in `.claude/skills/creative-lead/references/`

## Testing
Test with real historical briefings from Asana. Compare output against predecessor's actual work.
Quality gate: >80% match with human Creative Lead output.

## Clients (Pilot)
Hisense Deutschland, Gorenje Deutschland, SIXT (plus new pitches like MINI)
